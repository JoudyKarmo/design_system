usu-accordion

Encapsulation: shadow

The Accordion should always wrap Accordion Panels.

Properties​

PropertyAttributeDescriptionTypeDefault
multiplemultipleWhether the accordion should allow multiple expanded accordion panels simultaneously.booleanfalse

Methods​

usuCloseOtherPanels​

usuCloseOtherPanels(panel: HTMLUsuAccordionPanelElement) => Promise<void>

Collapses other accordion panels

Returns​

Type: Promise<void>

usuExpandPanel​

usuExpandPanel(panel: HTMLUsuAccordionPanelElement) => Promise<void>

Expands a panel of the accordion

Returns​

Type: Promise<void>

usuExpandPanelAbove​

usuExpandPanelAbove(panel: HTMLUsuAccordionPanelElement) => Promise<void>

Opens a panel directly on top of the @param panel if there is any
This is used if some panel that is opened is removed from the accordion

Returns​

Type: Promise<void>

Slots​

SlotDescription
The content of the accordion
